LS: Para enumerar lo que hay en el directorio actual

npm init -y: Sirve para instalar y crear el paquete Json inicial
npm install vite: Para instalar una bibloteca de dependencias de https://www.npmjs.com/
npm install three; Sirve para instalar las dependencias de Three.js
