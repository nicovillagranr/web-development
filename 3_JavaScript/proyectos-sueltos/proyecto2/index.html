JavaScript es un lenguaje de programación de alto nivel que se utiliza para hacer las páginas web interactivas y dinámicas. Es una de las tres tecnologías principales de la web, junto con HTML y CSS, y permite realizar acciones como crear animaciones, validar formularios y actualizar contenido sin necesidad de recargar la página. Funciona directamente en el navegador del usuario, ejecutándose el código allí mismo.
